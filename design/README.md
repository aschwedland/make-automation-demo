# Design Handoff — AI Lead Follow-Up Automation (Case Study Page)

The design for this project lives in Claude Design. Use the `claude_design` MCP to access it directly in Claude Code — no zip file needed.

## How to Access the Design

1. In Claude Code, run `/design-login` to authenticate with Claude Design
2. Then paste this prompt to import the project:

```
Use the claude_design MCP (https://api.anthropic.com/v1/design/mcp, auth via /design-login) to import this project: https://claude.ai/design/p/68b5b682-388b-4abb-a09f-3483505bb5d3?file=Case+Study.dc.html

Implement: Case Study.dc.html
```

## What's in the Design

Two frames:
- **Desktop** — full page scroll, all 6 sections (hero, problem, automation flow, video placeholder, what it replaces, CTA)
- **Mobile (375px)** — hero, automation flow (vertical steps), CTA section

Visual system: white bg, blue accent (#2563EB), Inter font, max-width 960px centered. The 4-step automation flow diagram is the key design element.
